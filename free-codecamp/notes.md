# Basic JavaScript

## Boolean

- two values: `true` or `false`
	- never written with quotes

## Conditonal logic

```js
if (condition is true) {
    statement is executed;
}
```

- equality operator: `= =`
	- Typecase string and number automatically
- strict equality operator: `= = =`
- inequality operator: `! =`
- strict inequality operator: `! = =`
- `&&` and
- `||` or
