## Script Tag

```html
<body>
  <h1 onclick="sayHello();">
    Hello
  </h1>
</body>

<script type="text/javascript">
	function sayHello(){
    alert('hello');
  };
</script>
```

## Anonymous functions

```javascript
function sayGoodbye(){
	$("#title").html("Goodbye");
	$("#title").click(function(){
		$("#title").html("Hello");
		$("#title").off("click");
	});
}
```
