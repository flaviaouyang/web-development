# Elements

```HTML
<!-- specifies the base URL to use for all relative URLs in a document -->
<base>

<!-- enclosed HTML provides contact information for a person or people, or for an organization -->
<address>

<!-- a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable -->
<article>

<!-- a portion of a document whose content is only indirectly related to the document's main content -->
<aside>

<!-- quote -->
<blockquote>

<!-- definition -->
<dl>
  <dt>Coffee</dt>
  <dd>Black hot drink</dd>
  <dt>Milk</dt>
  <dd>White cold drink</dd>
</dl>

<!-- image caption -->
<figure>
  <img>
  <figcaption></figcaption>
</figure>

<!-- semantic alternative to unordered lists -->
<menu>

<!-- an area inside an image map that has predefined clickable areas -->
<area>

<!-- embed sound -->
<audio>

<!-- define a map -->
<map>

<!-- external content at the specified point in the document -->
<embed>

<!-- a nested browsing context, embedding another HTML page into the current one -->
<iframe>

<!-- n external resource, which can be treated as an image, a nested browsing context, or a resource to be handled by a plugin -->
<object>

<!-- enables the embedding of another HTML page into the current one for the purposes of allowing smoother navigation into new pages -->
<portal>

<!-- a container that defines a new coordinate system and viewport -->
<svg>

<!-- with either the canvas scripting API or the WebGL API to draw graphics and animations -->
<canvas>
```
