# Condition

## Comparisons

- greater / lesser than: `a > b`, `a < b`
- Greater / lesser than or equal to: `a >=b, a <= b`
- Equal: `===`
- not equal: `!=!`

## if statement

```javascript
if (condition) {
  // ...
} else if (condition) {
  // ...
} else {
  // ...
}
```

## logical operators

- `||`: or
- `&&`: and
- `!`: not
- `??`: nullish coalescing

## Ternary operator

```javascript
( conditon ) ? run this code : run this code instead

let greeting = ( isBirthday ) ? 'Happy BDay' : 'cool';
```
